# Icepi CM0

A low-cost carrier board for the H3 CPU. It let's you have a cheap yet powerful Linux/Android device on your hand.

![](https://hc-cdn.hel1.your-objectstorage.com/s/v3/65acf57737127a090191e2a362e5340783381ce1_image.png)

## Features

- H3 Processor
    - Quad-Core ARM Cortex-A7 @ 1.296GHz
    - ARM Mali400 MP2 OpenGL ES 2.0
- 512MiB of DDR3 RAM
- Ethernet PHY
- WiFi & Bluetooth
- GPDI
- Raspberry-pi compatiable GPIO
- USB connectors:
    - 2x USB-A 
    - 2x USB-C with one OTG
    - 1x USB-C 100W PD
- 3.5mm audio jack
- SD card slot
- eMMC 5.1 storage
